# minwage-rent-burden

**Do Minimum Wage Increases Improve Housing Affordability?**
A state-level panel analysis (2010–2024) using two-way fixed effects difference-in-differences.

> **Report: [English](report/report.md) | [中文](report/report_zh.md) | [日本語](report/report_ja.md)**

---

## Research Question

Do state minimum wage increases reduce the share of income that renters spend on housing?

Using a balanced panel of 51 US states/DC over 2010–2024, we exploit staggered timing of state minimum wage increases (≥$0.50 threshold) relative to the federal floor ($7.25) to estimate causal effects on rental affordability via a TWFE DiD design, augmented by event study analysis and robustness checks.

## Key Findings

| Result | Estimate | p-value |
|--------|----------|---------|
| Rent burden (% of income) after MW increase | +0.22 pp | 0.210 |
| Absolute rent level after MW increase | +$64/month | **0.003** |
| Continuous MW gap → rent burden | −0.015 pp/$ | 0.711 |

- **No significant improvement** in housing affordability (rent-to-income ratio) following minimum wage increases
- **Significant increase in absolute rent levels** (+$64/month), suggesting landlord cost pass-through
- Event study shows **no pre-trend violations**, supporting causal interpretation
- Rent levels and wage gains roughly offset, leaving the affordability ratio unchanged

## Reproduce

### 1. Environment Setup

```bash
# Python 3.11+ required
bash scripts/setup_env.sh
source .venv/bin/activate
```

### 2. Download Data & Run Full Pipeline

```bash
bash scripts/run_pipeline.sh
```

This executes 10 stages sequentially:

1. **Download** — Fetch raw data from FRED, Census ACS, BLS LAUS, BLS QCEW
2. **Clean policy** — Construct minimum wage variables, identify treatment timing (≥$0.50 threshold)
3. **Clean outcomes** — Process ACS rent burden and gross rent
4. **Clean controls** — Process unemployment rate and weekly wages
5. **Build panel** — Merge into balanced 51×15 state-year panel
6. **EDA** — Summary statistics, balance table, trend plots
7. **Baseline DiD** — Two-way fixed effects regressions (3 specifications)
8. **Event study** — Dynamic effects for rent burden and log rent
9. **Robustness** — Alternative specifications, placebo permutation test
10. **Figures** — Coefficient plots, event study plots, diagnostics

### 3. Skip Download (if data already exists)

```bash
bash scripts/run_pipeline.sh --skip-download
```

### 4. Include Zillow Home Value Data (optional)

```bash
bash scripts/run_pipeline.sh --with-zillow
```

## Data Sources

All data is downloaded programmatically from public APIs. See [DATA_SOURCES.md](DATA_SOURCES.md) for complete provenance.

| Source | Variable | Provider |
|--------|----------|----------|
| FRED | State & federal minimum wage | Federal Reserve Bank of St. Louis |
| ACS B25071 | Median gross rent as % of income | U.S. Census Bureau |
| ACS B25064 | Median gross rent ($) | U.S. Census Bureau |
| BLS LAUS | State unemployment rate | Bureau of Labor Statistics |
| BLS QCEW | Average weekly wage | Bureau of Labor Statistics |
| Zillow ZHVI | Home value index (optional) | Zillow Research |

**Optional API keys** (increase rate limits):

```bash
export CENSUS_API_KEY="your_key"   # https://api.census.gov/data/key_signup.html

```

## Methodology

- **Design**: Two-way fixed effects (state + year FE) difference-in-differences
- **Treatment**: States raising minimum wage ≥$0.50 in a single year (filters CPI micro-adjustments)
- **Inference**: OLS with state-clustered standard errors (51 clusters)
- **Sample**: 714 state-year observations (51 units × 14 years, excluding 2020)
- **Robustness**: Continuous treatment intensity, state linear trends, pre-2020 subsample, alternative outcome, alternative treatment definition, placebo permutation test

## Note on QCEW

The BLS QCEW endpoint may reject automated requests. If the download fails, the pipeline continues without the wage control variable (the `available_controls` function requires ≥50% non-missing coverage to include a variable). To manually add QCEW data, place the file at `data/raw/qcew_state_avg_weekly_wage_2010_2024.csv` and re-run:

```bash
bash scripts/run_pipeline.sh --skip-download
```
